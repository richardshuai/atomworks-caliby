Tutorial
========

Getting Started
---------------

**cifutils** is a Python toolkit for parsing, converting, and manipulating biological structure and sequence data. It provides a unified interface for working with mmCIF, PDB, FASTA, SMILES, and more, enabling seamless conversion and preparation for downstream analysis or machine learning. Built on top of Biotite, it offers robust parsing, annotation, and transformation utilities for protein, nucleic acid, and small molecule data.

Quick Install:

.. code-block:: bash

   pip install cifutils  # or see Installation section for more options
